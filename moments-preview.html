v4 调整（按你反馈）

① 发现页 3 入口：朋友圈 + 收藏 + 日记。收藏 / 日记先做 UI 占位（点进去"敬请期待"），朋友圈先实现。
② 朋友圈主页顶部 2 按钮：
  • 齿轮 → 朋友圈设置弹窗（按图3）
  • 相机 → 发表朋友圈（点开发布器）
  • 去掉刷新（下拉刷新）和通知（移到朋友圈设置里）
③ 签名可编辑：点签名区域 → 弹输入框（沿用 FullScreenEditor v2 风格）→ 保存到 localStorage `os_moments_signature`
朋友圈主页 v4 关键改动（实现注意）：
  • 封面图 + 名字/签名/头像 = 一个滚动单元，不 sticky，跟列表一起滚
  • 顶部工具栏（左箭头 + 齿轮 + 相机）= 独立 sticky，永远在顶部
  • 头像形状 = 读 OSContext 的 `theme.bubbleStyle`（已有的"方/圆"主题设置）
  • 签名可编辑：点签名 → 弹 FullScreenEditor v2 编辑 → localStorage 存 `os_moments_signature`
  • 封面图长按 → 选图 / 重置
  • 朋友圈设置弹窗（按图3）—— 4 个 toggle + 手动生成区
这一轮（朋友圈）的范围：
  • 发现页（3 入口 UI，朋友圈可用）
  • 朋友圈主页（封面图 + 名字/签名/头像 + 列表）
  • 朋友圈设置弹窗（按图3）
  • 发布器
  • 动态详情 + 评论
  • 用户发朋友圈 + 数据存储（localStorage）
  • 签名可编辑（localStorage）
  • 封面图长按换图
  • 头像跟随主题方圆
下一轮（朋友圈增强）：
  • AI 主动发朋友圈（LLM 自由决定 + schedule 触发）
  • AI 自动评论/点赞（用户发朋友圈后 LLM 互动）
  • 生图 API 集成（角色配图）
  • 本地通知（AI 发朋友圈时）
后续轮：
